# APEXIS — Construction Communication Infrastructure

Apexis is a smart construction storage, documentation, and reporting platform built for architects, designers, consultants, builders, and project owners.

## Technology Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
npm i
npm run dev
```
